### Prerequisites
- Modern web browser
- Basic understanding of HTML/CSS/JavaScript
- Text editor or IDE for modifications

### Installation

1. **Clone the repository**
   ```bash

   cd Frontend-RoofingCompany
   ```

2. **Open in browser**
